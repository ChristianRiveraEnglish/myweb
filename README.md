# Sitio de Christian Rivera Gavidia

## Cómo publicarlo (resumen)
1. Sube esta carpeta completa a un repositorio nuevo en GitHub (github.com, gratis).
2. Entra a vercel.com, "Add New Project", conecta tu cuenta de GitHub y selecciona este repositorio.
3. Vercel detecta Vite automáticamente. Click "Deploy". Listo, ya está publicado.
4. Para tu dominio propio: en Vercel, ve a Project Settings > Domains, escribe tu dominio,
   y sigue las instrucciones para apuntar los DNS desde donde compraste el dominio.

## Para editar después
Cualquier cambio en src/App.jsx y volver a subirlo a GitHub actualiza el sitio automáticamente.
